# market-data-lookup

## Getting Started

### Checkout
    
    git clone https://github.com/Niederegger/market-data-lookup.git
    
### Build

    cd market-data-lookup
    mvn clean package
    
### Startup Server

    cd target
    java -jar demo-0.0.1-SNAPSHOT.jar
    

### Usage

Start browser and visit `http://localhost:9090/lookup?isin=A000XXXXX`
